The Library Management System is designed to:

- Manage book records

- Register and manage students

- Issue and return books

- Track borrowing history

- Reduce manual paperwork

- Improve accuracy and efficiency

This project demonstrates backend development skills using PHP along with frontend technologies.

---

Technologies Used

**Frontend**:

HTML

CSS

JavaScript

**Backend**:

PHP

**Database**:

MySQL

**Server**:

XAMPP

---
**Admin** **Module**

Add new books

Update book details

Delete books

View all books

Manage student records

Issue books to students

Accept returned books

View issued book history

 **User** **Module**

View available books

Search books by title/author

View issued books

Check due dates
